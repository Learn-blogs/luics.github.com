数据

  • 淘宝无线数据指数 (WDM)：数据来自阿里巴巴90%的App
  • 友盟指数：数据来自10w+ App，App统计市场占有率第一

数据：OS


数据：屏幕分辨率

  • MDN Viewport | Viewport size
  • Android 简化至： 480/320/540/240系、640/720/800系

数据：浏览器

MGBS

MGBS落地

无论是OS版本、分辨率分类、还是浏览器版本 我们的目标是用最少开发代价，覆盖满足业务需求的终端设备集 MGBS被天猫和其他兄弟公司所引用，落地到业务会有更明确

MGBS 遗留问题

  • ipad retina (屏宽1536) 支持
  • android 屏宽 720/800 支持
  • 所有移动设备的横屏支持
  • android 2.3.x与4.x的默认浏览器差异，以及可接受的退化处理范围
  • android qq浏览器是否仍然在使用的WebView，是否有必要放在A级
  • android qq浏览器有独立和内置两种形式，是否有差异
  • windows phone 支持
